# TG Notion

Заметки и задачи прямо в Telegram Mini App.

## Что умеет

- 📝 Заметки — создавай, открывай, удаляй
- ✅ Задачи — простой to-do лист, нажал и зачеркнул
- 🌗 Тёмная и светлая темы
- 💾 Данные хранятся локально (localStorage) (ВРЕМЕННО!)

## Как работает

Mini App открывается через бота [@Telega_notion_bot](https://t.me/Telega_notion_bot). Бот — входная дверь, всё остальное делает Mini App.

Хостинг — GitHub Pages. Бесплатно, быстро, без сервера.(ВРЕМЕННО!!!)

## Что дальше

- Синхронизация между устройствами
- Шеринг заметок и задач
- Общие доски
- Напоминания
- Категории и папки
